  "редактировать книгу" - изменить поля книги;
  "печать книги" - распечатать сведения интересующей книги;
  "переименовать книгу" - изменить наименование книги в текущем разделе;
  "удалить книгу" - удаление книги
  "создать подраздел" - cоздание нового раздела в текущем для хранения книг;
  "открыть подраздел" - перейти в подраздел текущего раздела;
  "раздел выше" - открыть раздел выше по иерархии;
  "переименовать раздел" - изменить наименование текущего раздела;
  "показать подразделы" - показать все подразделы текущего раздела;
  "удалить подраздел" - удалить подраздел (со всем его содержимым);

  "установить глобальное поле" - изменить поля книги;
  "показать глобальные поля" - показать все существующие глобальные поля;
  "снять глобальное поле" - убрать глобальное значение поля;
  "поиск книги" - поиск книг в системе в соответствии с параметрами поиска;
  "сохранить", "сохранение", "save" - сохранить систему для учета книг.
  "сохранить как" - сохранить систему для учета книг по новому местоположению.

2) выйти из системы для учета книг, введя: "выход", или "выйти", или "exit", или "leave".
Для выведения списка команд, введите "команды".
